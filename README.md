# Driftlog
Where tasks find their current.

## Setup & Run

### Backend
\`\`\`bash
cd server
npm install
node index.js
\`\`\`

### Frontend
\`\`\`bash
cd client
npm install
npm start
\`\`\`

### Open
http://localhost:3000

## Features
- **Task Management**: Create, Read, Update, Delete tasks with priorities and statuses.
- **Dashboard**: Visual progress bars and Chart.js metrics tracking your productivity.
- **Authentication**: JWT-based login and registration.
- **Dark Mode**: Toggleable beautiful dark theme built with CSS variables.
